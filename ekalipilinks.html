Ekalipi Keyboard (Windows)
  • Download & Install


Download and Install

  1. Download the Installation package. Save the file at the default location (or at a different location, if you are familiar with Windows).
  2. Navigate to the file location and double-click on the file name Ekalipi6.4.exe.
  3. Save this Ekalipi6.4.exe file.
  4. Click on the saved executable (i.e. .exe) file. Note- You may receive the following warning. Do not be concerned. This is the standard Windows warning for .exe files. Click on "Run Anyway".
  5. Click on the Yes button.
  6. You should now see the "installation complete" message window. This means that the Ekalipi Keyboard has been installed on your computer.
  7. Note - If you are using Windows 7, please see the Windows 7 instructions instead.
    8. Activating the Ekalipi keyboard.

Keyboard Setup for Windows 10

  1. Go to Language Settings, click on the Add Language button.
  2. On the left side, you should see "Ekalipi Script Keyboard-Custom". When you click that option, the Ekalipi Keyboard is added in your Language section.
  3. Reboot your computer. After you have rebooted the computer, click on the keyboard icon on the taskbar. You will first see the default keyboard.
  4. Click on the keyboard icon in the right hand corner then click on the Ekalipi keyboard. You should now see the Ekalipi keyboard:
    5. CONGRATULATIONS! You have successfully installed and activated the Ekalipi keyboard. From this point onwards, you can switch keyboards by clicking on the language button in your taskbar.

Keyboard Setup for Windows 7

    Keyboard Setup for Windows 7
  • Click window+r button
  • Type "ctfmon" in open section.Click "ok".
  • After that star the "on screen keyboard".
  • On the right side of the taskbar, you will see the EN icon.
  • Click on this icon, you should see "Ekalipi Script Keyboard-Custom".
  • Tap on it, it will show the Ekalipi Windows Keyboard.
  • If you do not see the Ekalipi Keyboard go to Control Panel- Region and Language. Click on Keyboards and Languages. Then click on the Change Keyboards button.
  • After that open Text Services and Input Languages window. You will see "Ekalipi Script Keyboard-Custom", click on that keyboard then click on the OK or Apply button.
  • Click on Change Key Sequence.. Tick the Enable Key Sequence Left Alt + Shift-- Key 1, then click Ok button.
